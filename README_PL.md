# AR-APP — wersja jednoplikowa

Ta wersja zawiera wszystko w jednym pliku `index.html`.

Nie trzeba wgrywać folderu `assets`.
Nie trzeba wgrywać obrazów referencyjnych osobno.
Podmień tylko `index.html` w repozytorium GitHub.

Instrukcja:

1. Otwórz repozytorium AR-APP na GitHub.
2. Kliknij plik `index.html`.
3. Kliknij ikonę ołówka.
4. Usuń całą starą treść.
5. Wklej zawartość nowego `index.html`.
6. Kliknij `Commit changes`.
7. Poczekaj 1–2 minuty.
8. Otwórz: https://aangelikaa1998.github.io/AR-APP/
